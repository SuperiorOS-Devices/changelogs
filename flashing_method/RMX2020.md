# Method:

* Clean flash: 
1. Flash recovery.img from PBRP
2. Boot to recovery
3. Flash the ROM zip
4. Format data
5. Boot to system

* Dirty flash: 
1. Clean Dalvik and Cache
2. Flash ROM zip
3. Reboot system
